# Мой Зал 🏋️ — дневник тренировок для iPhone

Приложение для отслеживания тренировок в тренажёрном зале. Работает как PWA:
устанавливается на экран «Домой» iPhone, выглядит и работает как обычное
приложение, не требует App Store и интернета — все данные хранятся на телефоне.

> **Нативная версия (SwiftUI)** — в папке [`ios/`](ios/README.md). Она нужна
> только ради двух вещей, которые веб на iOS не умеет: уведомление об окончании
> отдыха приходит на Apple Watch даже при закрытом приложении, а данные из
> «Здоровья» (тренировки, пульс, калории, вес) читаются автоматически через
> HealthKit. Собирается в Xcode на Mac.

## Возможности

- **Тренировка** — начни тренировку, добавляй упражнения, записывай подходы
  (вес × повторы). Приложение показывает результаты прошлого раза и само
  подставляет их в новые подходы.
- **Готовые программы** — 6 мужских и 6 женских программ (фулбоди, верх/низ,
  жим/тяга/ноги, сила 5×5, сплиты, ягодицы и ноги, круговые). Любой день
  программы запускается в тренировку одним тапом.
- **Свои программы** — конструктор: собери программу из любых упражнений
  с нужными подходами и повторами, редактируй и запускай.
- **Таймер отдыха** — после отметки подхода автоматически запускается обратный
  отсчёт (длительность настраивается), со звуком по окончании.
- **База упражнений** — 45 упражнений по группам мышц + свои собственные.
- **Видеоинструкции** — кнопка ▶ у каждого упражнения открывает видео
  с техникой выполнения.
- **История** — все тренировки с длительностью, тоннажем, заметками и списком
  подходов; любую тренировку можно повторить одним тапом.
- **Личные рекорды** — приложение само замечает новый лучший вес и поздравляет
  при завершении тренировки; рекорды видны на графике каждого упражнения.
- **Прогресс** — графики по каждому упражнению (вес / расчётный 1ПМ / объём),
  тоннаж по неделям, серия недель без пропусков, график веса тела, статистика.
- **Блины на штангу** — калькулятор: какие блины повесить с каждой стороны
  грифа под целевой вес.
- **Экран не гаснет** во время активной тренировки (Wake Lock).
- **ИИ-тренер** — чат с бесплатной ИИ-моделью, которая видит твои последние
  тренировки, рекорды и вес: составит тренировку, разберёт прогресс, подскажет
  как пробить плато. Работает через собственный бесплатный ключ OpenRouter
  (без карты, до 50 запросов/день) или Google Gemini (бесплатный тариф);
  ключ хранится только на телефоне.
- **Apple Watch** — импорт файла экспорта из приложения «Здоровье»
  (Здоровье → профиль → Экспорт медданных → export.xml): тренировки с часов
  с метриками (ккал, средний и максимальный пульс, дистанция) попадают
  в историю, замеры веса — в график веса тела; в «Прогрессе» — сводка
  «Здоровье за 4 недели». Прямой доступ к HealthKit веб-приложениям
  Apple не предоставляет.
- **Уведомление об окончании отдыха** — системное уведомление со звуком;
  на заблокированном iPhone iOS дублирует его на Apple Watch. Включается
  в «Ещё → Настройки», работает в установленном на экран «Домой» приложении.
- **Калькулятор КБЖУ** — калории и макронутриенты по формуле
  Миффлина — Сан-Жеора под цель (похудение/поддержание/набор).
- **Калькулятор 1ПМ** — расчёт одноповторного максимума по формуле Эпли
  и таблица процентов для программ.
- **Книга рецептов** — 39 ПП-блюд по программе питания (завтрак, обед, ужин,
  десерт) с КБЖУ, ингредиентами и пошаговым приготовлением; выбор блюд с поиском
  по названию и ингредиенту и фильтром по категориям.
- **База знаний** — 12 статей: с чего начать, прогрессия нагрузок, подходы
  и повторы, питание, восстановление, кардио, мифы.
- **Экспорт/импорт** — резервная копия всех данных в JSON-файл.

## Как установить на iPhone

1. **Опубликуй сайт.** Самый простой способ — GitHub Pages:
   - Открой настройки репозитория: **Settings → Pages**.
   - В разделе **Build and deployment → Source** выбери **GitHub Actions**.
   - Смёржи этот код в ветку `main` (или нажми **Run workflow** у
     «Deploy to GitHub Pages» во вкладке Actions).
   - Через минуту приложение будет доступно по адресу
     `https://<твой-логин>.github.io/<имя-репозитория>/`.
2. **Открой этот адрес в Safari** на iPhone.
3. Нажми кнопку **Поделиться** (квадрат со стрелкой) → **На экран «Домой»** →
   **Добавить**.
4. Готово: на рабочем столе появится иконка «Мой Зал». Запускай из неё —
   приложение откроется на весь экран и будет работать даже без интернета.

## Где хранятся данные

Все данные лежат локально на телефоне (localStorage) и никуда не отправляются.
Чтобы не потерять их при смене телефона или очистке Safari, время от времени
делай **Экспорт данных** на вкладке «Прогресс».

## Технологии

Чистые HTML/CSS/JavaScript без зависимостей и сборки. Service worker кэширует
приложение для офлайн-работы. Открой `index.html` через любой статический
сервер — и всё работает.
