# Мой Зал 🏋️ — дневник тренировок для iPhone

Приложение для отслеживания тренировок в тренажёрном зале. Работает как PWA:
устанавливается на экран «Домой» iPhone, выглядит и работает как обычное
приложение, не требует App Store и интернета — все данные хранятся на телефоне.

## Возможности

- **Тренировка** — начни тренировку, добавляй упражнения, записывай подходы
  (вес × повторы). Приложение показывает результаты прошлого раза и само
  подставляет их в новые подходы.
- **Таймер отдыха** — после отметки подхода автоматически запускается обратный
  отсчёт (длительность настраивается), со звуком по окончании.
- **База упражнений** — 45 упражнений по группам мышц (грудь, спина, ноги,
  плечи, руки, пресс, кардио) + свои собственные.
- **История** — все тренировки с длительностью, тоннажем и списком подходов.
- **Прогресс** — графики роста рабочего веса по каждому упражнению, график
  веса тела, общая статистика.
- **Экспорт/импорт** — резервная копия всех данных в JSON-файл.

## Как установить на iPhone

1. **Опубликуй сайт.** Самый простой способ — GitHub Pages:
   - Открой настройки репозитория: **Settings → Pages**.
   - В разделе **Build and deployment → Source** выбери **GitHub Actions**.
   - Смёржи этот код в ветку `main` (или нажми **Run workflow** у
     «Deploy to GitHub Pages» во вкладке Actions).
   - Через минуту приложение будет доступно по адресу
     `https://<твой-логин>.github.io/<имя-репозитория>/`.
2. **Открой этот адрес в Safari** на iPhone.
3. Нажми кнопку **Поделиться** (квадрат со стрелкой) → **На экран «Домой»** →
   **Добавить**.
4. Готово: на рабочем столе появится иконка «Мой Зал». Запускай из неё —
   приложение откроется на весь экран и будет работать даже без интернета.

## Где хранятся данные

Все данные лежат локально на телефоне (localStorage) и никуда не отправляются.
Чтобы не потерять их при смене телефона или очистке Safari, время от времени
делай **Экспорт данных** на вкладке «Прогресс».

## Технологии

Чистые HTML/CSS/JavaScript без зависимостей и сборки. Service worker кэширует
приложение для офлайн-работы. Открой `index.html` через любой статический
сервер — и всё работает.
