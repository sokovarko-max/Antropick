# Dashboard Pro — умная боковая панель для Windows 11

Современный «Центр управления»: боковая панель, закреплённая у правого края экрана,
которая собирает в одном месте время, погоду, календарь, задачи, заметки, привычки,
проекты, Pomodoro и подсказки ассистента.

**Стек:** C# / .NET 8 · WinUI 3 · Windows App SDK 1.5 · SQLite

## Возможности

- **Док у правого края** — панель всегда прижата к правому краю монитора на всю высоту.
- **Поверх всех окон** — переключается кнопкой-булавкой или в настройках.
- **Резервирование места (AppBar)** — как у панели задач: развёрнутые окна не
  перекрываются панелью (Shell AppBar API).
- **Дизайн Windows 11** — Mica / Mica Alt / Acrylic, скругления, Segoe Fluent Icons,
  светлая и тёмная темы, настраиваемая прозрачность.
- **Автозапуск** при входе в Windows (ключ реестра `Run`).
- **Глобальный поиск** по задачам, событиям, заметкам и проектам.

### Модульная система

Каждый модуль — карточка, которую можно:

- убрать с панели (✕) и вернуть (кнопка «+» в шапке);
- перетащить на другое место (drag-and-drop, порядок сохраняется);
- свернуть/развернуть;
- настроить через шестерёнку на карточке (источник, ключи API, интервалы).

| Модуль | Что показывает |
| --- | --- |
| 🕒 Время | Часы и дата |
| 🌤 Погода | OpenWeather или WeatherAPI |
| 📅 Сегодня | События календаря на сегодня |
| ⭐ Главные задачи | Просроченные, важные и «горящие» задачи + быстрый ввод |
| 🗓 План дня | Хронология: встречи + задачи с дедлайном сегодня |
| 🍅 Pomodoro | Настраиваемые циклы, автостарт, уведомления, статистика в SQLite |
| 📝 Заметки | Быстрые заметки (база приложения или папка Markdown/Obsidian) |
| 💪 Привычки | Отметка на сегодня + последние 7 дней |
| 📈 Проекты | Прогресс, дедлайн, количество задач, следующий шаг |
| 💻 Система | CPU, память, скорость сети |
| 💡 Ассистент | «До встречи осталось 42 минуты», «Просроченных задач: 3» и т.п. |

### Источники данных

Настраиваются в окне настроек (⚙ в шапке панели):

- **Задачи:** база приложения (SQLite) · Markdown-файл или папка (формат
  Obsidian `- [ ] задача due:2026-08-01 !`) · CSV · **Todoist** (REST API v2, по токену).
- **Календарь:** любая ICS-лента — секретная ссылка Google Calendar,
  «Опубликовать календарь» Outlook / Microsoft 365, локальный `.ics` файл.
- **Заметки:** база приложения · папка Markdown/TXT (Obsidian-vault).
- **Погода:** OpenWeather · WeatherAPI (ключ и город — в настройках модуля).

### Плагины

Интеграции подключаются без изменения ядра: DLL в
`%LOCALAPPDATA%\DashboardPro\Plugins` с классом, реализующим `IDashboardPlugin`:

```csharp
public sealed class NotionPlugin : IDashboardPlugin
{
    public string Name => "Notion";

    public void Register(ISourceRegistry registry)
    {
        registry.RegisterTaskSource("notion", s => new NotionTaskSource(s["token"], s["databaseId"]));
    }
}
```

После этого в `settings.json` достаточно указать `"TaskSource": { "type": "notion", … }`.
Аналогично расширяются календари, заметки и погода. Интерфейс `IInsightProvider`
позволяет заменить встроенный движок подсказок на LLM-провайдера.

## Сборка

Требуется Windows 10 17763+ / Windows 11 и Visual Studio 2022 (17.9+) с нагрузкой
«Разработка классических приложений .NET» + компоненты Windows App SDK, либо .NET 8 SDK:

```powershell
dotnet build DashboardPro.sln -p:Platform=x64
dotnet run --project src/DashboardPro.App -p:Platform=x64
```

Приложение неупакованное (`WindowsPackageType=None`), Windows App SDK включён
самодостаточно — MSIX и установка рантайма не нужны.

Данные и настройки: `%LOCALAPPDATA%\DashboardPro\` (`settings.json`, `dashboard.db`).

## Архитектура

```
DashboardPro.sln
└── src
    ├── DashboardPro.Core          — чистый .NET 8, без UI-зависимостей
    │   ├── Models                 — TaskItem, CalendarEvent, Note, Habit, Project, …
    │   ├── Services               — SettingsService (JSON), LocalStore (SQLite), InsightEngine
    │   └── DataSources            — контракты ITaskSource/ICalendarSource/INoteSource/IWeatherProvider,
    │                                реализации (Markdown, CSV, Todoist, ICS, OpenWeather, WeatherAPI),
    │                                SourceFactory + загрузчик плагинов
    └── DashboardPro.App           — WinUI 3 (Windows App SDK)
        ├── Interop                — док к краю экрана, Shell AppBar, DPI
        ├── Services               — DataHub (общий кэш данных), автозапуск, системный монитор
        ├── Controls/ModuleHost    — карточка модуля: заголовок, свернуть/закрыть/настройки
        └── Modules                — 11 модулей + реестр ModuleRegistry
```

Ключевые решения:

- **DataHub** — единый кэш: источники опрашиваются по расписанию один раз,
  модули получают событие `Updated` и рендерятся из кэша.
- **ModuleRegistry** — декларативный список модулей: id, заголовок, иконка,
  фабрика контрола и схема настроек (по ней автоматически строится UI настроек модуля).
- Порядок, видимость, свёрнутость и настройки каждого модуля хранятся в `settings.json`.

## Ограничения v1 / план

- RRULE (повторяющиеся события) в ICS пока не разворачиваются.
- AppBar не обрабатывает `ABN_POSCHANGED` (смена разрешения — переоткрыть панель или нажать «Сохранить» в настройках).
- OAuth-интеграции (Microsoft To Do, Google Tasks, Notion, Jira, Trello, …) — как плагины.
- В планах: трей-иконка, буфер обмена с историей, быстрый запуск программ,
  голосовые команды, почтовый модуль, LLM-ассистент.
